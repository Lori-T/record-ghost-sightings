This page lets you jump to any screen in the Record a ghost sighting prototype. Use it to explore or test specific accessibility issues without completing the full journey.

Pages with intentional accessibility issues

  • Start page — missing title, low contrast, no alt text
  • Where did you see the ghost? — label and focus issues
  • What did the ghost look like? — missing alt text, poor contrast
  • When did you see the ghost? — incorrect fieldset, reflow issue
  • Did the ghost interact with you? — keyboard and conditional issues
  • Check your answers — duplicate link text, table structure issues
  • Confirmation page — heading and resize issues

Testing ideas

  • Try using only the keyboard to move through the journey
  • Zoom in to 400% and check layout and reflow
  • Use a screen reader to identify missing labels or names
  • Check colour contrast for text and buttons
  • Look for inconsistent or unclear link purpose
